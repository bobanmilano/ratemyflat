# Konto und Daten löschen für RateMyFlat

Sie können Ihr Konto und die damit verbundenen Daten in der RateMyFlat-App jederzeit löschen.

## So löschen Sie Ihr Konto:

1.  Öffnen Sie die RateMyFlat-App.
2.  Gehen Sie zu Ihren Kontoeinstellungen (meist über das Profil-Menü).
3.  Suchen Sie die Option "Konto löschen" oder "Account löschen".
4.  Bestätigen Sie die Löschung. Möglicherweise müssen Sie Ihr Passwort erneut eingeben.

## Was passiert bei der Kontolöschung?

*   **Gelöschte Daten:** Ihr Benutzerkonto (Benutzername, E-Mail-Adresse, Profilbild, Passwort) wird dauerhaft aus unseren Systemen entfernt.
*   **Bewertungen und Inhalte:** Bewertungen, Kommentare und Fotos, die Sie öffentlich erstellt haben, bleiben in der App sichtbar, werden aber anonymisiert (Ihr Benutzername wird z. B. durch "Anonymous" ersetzt). Dies geschieht, um die Integrität der Community-Bewertungen zu erhalten und anderen Nutzern weiterhin Informationen zu bieten. Rechtliche Gründe können eine vollständige Löschung erfordern.
*   **Verknüpfte Daten:** Verknüpfungen zwischen Ihren Bewertungen und Ihrem Konto werden entfernt.
*   **Aufbewahrung:** Technische Protokolldaten können aus Sicherheits- oder rechtlichen Gründen für eine begrenzte Zeit aufbewahrt werden.
